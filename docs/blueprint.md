# **App Name**: QuizWhiz

## Core Features:

- Dynamic Quiz Generation: Generates quizzes with questions pulled from the backend.
- Category and Difficulty Selection: Allows users to select a quiz category (e.g., Communication Skills, Aptitude) and difficulty level (Easy, Medium, Hard).
- Timed Quiz Engine: Implements a real-time quiz engine with dynamic timers for each difficulty level. Automatically submits the quiz when the time runs out.
- Performance Analytics: Provides detailed result and performance analytics after quiz completion, including score breakdown, charts, and question-wise analysis.
- Personalized Feedback: Utilizes generative AI to generate personalized feedback messages based on user performance in the quiz, providing tips and suggestions for improvement using quiz results as a tool.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of intellect and focus.
- Background color: Light gray (#F5F5F5) to ensure readability and a modern feel.
- Accent color: Bright yellow (#FFEB3B) for CTAs and important highlights.
- Headline font: 'Space Grotesk' sans-serif font for titles. Body font: 'Inter' sans-serif font for the quiz questions and result descriptions.
- Use clean, modern icons to represent quiz categories and results. All icons should be related to the test domain.
- Mobile-responsive layout with clear sections for quiz questions, timer, and results. Implement a modern design with glass-morphism style cards.
- Subtle animations for timers, transitions, and score reveals to enhance user engagement.
